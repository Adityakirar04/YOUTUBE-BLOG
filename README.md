 # 🎥 YouTube Blog

A modern full-stack blogging platform built with **Node.js**, **Express.js**, **MongoDB Atlas**, **EJS**, and **Bootstrap**. The application enables users to securely register, log in, create blog posts with cover images, and interact through a commenting system. It follows the **MVC (Model-View-Controller)** architecture and uses **JWT-based authentication** with cookies for secure session management.

The project demonstrates essential backend development concepts, including user authentication, CRUD operations, cloud database integration, file uploads using Multer, server-side rendering with EJS, and deployment on Render. It is designed with a responsive user interface, making it accessible across desktop and mobile devices.

## ⚙️ Installation

```bash
git clone https://github.com/Adityakirar04/YOUTUBE-BLOG.git

cd YOUTUBE-BLOG

npm install

npm run dev
```

Create a `.env` file:

```env
PORT=8000
MONGO_URL=YOUR_MONGODB_CONNECTION_STRING
JWT_SECRET=YOUR_SECRET_KEY
```

Visit:

```
http://localhost:8000
```
